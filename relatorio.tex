\section{Estrutura do SO}
O Freebsd utiliza uma hierarquia padrao unix, visivel em / bin,/ etc,/usr,/var.
-/bin:comandos essenciais.
-/etc:configuracoes do sistema.
-/usr:aplicativos dos usuarios.
-/var:logs e banco de dados (MySQL).
\section{Kernel}
O sistema utiliza o kernel monolitico FreeBSD 15.0-RELEASE.
